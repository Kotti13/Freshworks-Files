<!DOCTYPE html>
<html lang="en">

    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Basic HTML Tags</title>
</head>
<body>

<!-- Heading Tags -->
<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<h3>This is a Heading 3</h3>
<h4>This is a Heading 4</h4>
<h5>This is a Heading 5</h5>
<h6>This is a Heading 6</h6>

<!-- Paragraph Tag -->
<p>This is a paragraph. It can contain text, links, images, etc.</p>

<!-- Anchor Tag (Link) -->
<a href="https://www.example.com">Visit Example</a>

<!-- Image Tag -->
<img src="image.jpg" alt="Description of the image">

<!-- List Tags -->
<ul>
    <li>Unordered List Item 1</li>
    <li>Unordered List Item 2</li>
    <li>Unordered List Item 3</li>
</ul>

<ol>
    <li>Ordered List Item 1</li>
    <li>Ordered List Item 2</li>
    <li>Ordered List Item 3</li>
</ol>

<!-- Table Tag -->
<table border="1">
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
    <tr>
        <td>Row 2, Cell 1</td>
        <td>Row 2, Cell 2</td>
    </tr>
</table>

<!-- Form Tag -->
<form action="submit.php" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username"><br><br>
    
    <label for="password">Password:</label>
    <input type="password" id="password" name="password"><br><br>
    
    <input type="submit" value="Submit">
</form>

<!-- Division Tag -->
<div>
    <p>This is inside a division.</p>
</div>

<!-- Span Tag -->
<p>This is a <span style="color: blue;">blue</span> word.</p>

<!-- Comment Tag -->
<!-- This is a comment. -->

</body>
</html>







explanation:

<!DOCTYPE html>: Defines the document type and version of HTML being used.
<html>: The root element of an HTML page.
<head>: Contains meta-information about the document, such as character set, viewport settings, and title.
<title>: Sets the title of the document (displayed in the browser's title bar or tab).
<body>: Contains the content of the HTML document.
Headings (<h1> to <h6>): Used to define headings of varying levels.
Paragraph (<p>): Defines a paragraph of text.
Anchor (<a>): Creates hyperlinks to other web pages or resources.
Image (<img>): Embeds an image into the page.
Lists (<ul>, <ol>, <li>): Creates unordered (bullet points) and ordered (numbered) lists.
Table (<table>, <tr>, <th>, <td>): Defines a table with rows and columns.
Form (<form>, <input>, <label>, <textarea>, etc.): Creates a form for user input.
Division (<div>): Defines a division or section in an HTML document.
Span (<span>): Used to group inline-elements in a document for applying styles.
Comment (<!-- -->): Comments in HTML are not displayed in the browser and are used for developer notes.
